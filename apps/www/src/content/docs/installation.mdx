## 1. Installation

To get started with `@bun-ui/react`, install the library using your preferred
package manager:

```bash
# Using pnpm
pnpm add @bun-ui/react

# Using npm
npm install @bun-ui/react
```

## 2. Setting Up Styles

Bun UI provides two ways to use the components:

### Option 1: With Tailwind CSS (Recommended)

If you're using Tailwind CSS, you can integrate Bun UI's components with your existing Tailwind setup:

```js
// tailwind.config.js
module.exports = {
  content: [
    // other paths
    "../node_modules/@bun-ui/react/dist/**/*.{js,ts,jsx,tsx}",
  ],
  // other configurations
}
```

This approach allows you to:

- Customize components using Tailwind's utility classes
- Maintain consistent styling with your existing Tailwind setup
- Take advantage of Tailwind's optimization features

### Option 2: Without Tailwind CSS

If you're not using Tailwind CSS, you can use Bun UI's pre-built styles:

```jsx
// app/layout.tsx or entry point
import "@bun-ui/react/index.css"

function RootLayout({ children }: { children: React.ReactNode }) {
  return <>{children}</>
}
```

This approach:

- Requires no additional configuration
- Provides ready-to-use styled components
- Works with any React project

## 3. Example Usage

Once installed and configured, you can start using the components from the
library. For example:

```tsx
import { Button } from "@bun-ui/react"

function App() {
  return (
    <div>
      <Button variant="primary">Click Me</Button>
    </div>
  )
}

export default App
```
