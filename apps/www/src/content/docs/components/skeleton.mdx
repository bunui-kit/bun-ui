## Usage

Import the `Skeleton` component from `@bun-ui/react` and use it in your project:

```tsx
import { Skeleton } from "@bun-ui/react"
```

```tsx
<Skeleton width={100} height={100} />
```

## Examples

### Variants

The `Skeleton` component supports different variants such as `circle` and `rectangle`. Use the `variant` prop to specify the shape.

<ComponentPreview name="skeleton-variants" />

### Custom Dimensions

You can customize the width and height of the `Skeleton` component using the `width` and `height` props.

### Customization

The `Skeleton` component can be customized using the `className` prop or by extending its styles with your design system.

```tsx
<Skeleton
  width={200}
  height={50}
  className="bg-gray-300 animate-pulse"
/>
```

Refer to the source code for more advanced usage and customization options.
