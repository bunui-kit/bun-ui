## Usage

Import the `Slider` component from `@bun-ui/react` and use it in your project:

```tsx
import { Slider } from "@bun-ui/react"
```

```tsx
<Slider defaultValue={50}/>
```

### Range

The slider can be used to set the start and end of a range by
setting value of `isRange` to `true`. The `value` or `defaultValue`
must be an array of two numbers.
