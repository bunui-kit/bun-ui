## Usage

Import the `Tooltip` component from `@bun-ui/react` and use it in your project:

```jsx
import { Tooltip } from "@bun-ui/react"
```

```jsx
<Tooltip content="This is a tooltip">
  <button>Hover me</button>
</Tooltip>
```
