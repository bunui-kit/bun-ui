## Usage

The `Calendar` component is built on top of the [React DayPicker](https://daypicker.dev/) library.

Import the `Calendar` component from `@bun-ui/react` and use it in your project:

```tsx
import { Calendar } from "@bun-ui/react"
```

```tsx
<Calendar />
```
