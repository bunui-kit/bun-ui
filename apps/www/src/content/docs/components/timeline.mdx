## Usage

Import the Timeline components from `@bun-ui/react` and use them in your project:

```jsx
import {
  Timeline,
  TimelineItem,
  TimelineSeparator,
  TimelineIndicator,
  TimelineConnector,
  TimelineContent,
  TimelineTitle,
  TimelineDescription,
} from "@bun-ui/react"
```

```jsx
<Timeline>
  <TimelineItem>
    <TimelineSeparator>
      <TimelineIndicator />
      <TimelineConnector />
    </TimelineSeparator>
    <TimelineContent>
      <TimelineTitle>Event Title</TimelineTitle>
      <TimelineDescription>Event Description</TimelineDescription>
    </TimelineContent>
  </TimelineItem>
</Timeline>
```
