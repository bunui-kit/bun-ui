## Usage

Import the `Dialog` components from `@bun-ui/react` and use them in your
project:

```tsx
import {
  Dialog,
  DialogClose,
  DialogContent,
  DialogTitle,
  DialogTrigger,
} from "@bun-ui/react"
```

```tsx
<Dialog>
  <DialogTrigger className="btn">Open Dialog</DialogTrigger>
  <DialogContent>
    <DialogTitle>Dialog Title</DialogTitle>
    <p>This is the content of the dialog.</p>
    <DialogClose className="btn">Close</DialogClose>
  </DialogContent>
</Dialog>
```
